# Integrations

Vale Server works with a number of third-party applications. You'll find
installation and usage instructions for each plug-in or add-on below.

## Google Docs

```callout{'title': 'NOTE', 'classes': ['tip']}
The Vale Server add-on does not support the Safari web browser.
```

### Installation

Go to the [installation page]() and click the *Free* button:

<img alt="Google Docs menu" src="/vale-server/img/ui/Google/free.png" style="width: 100%; display: block; margin: 0 auto;">

Then, simply follow the on-screen instructions.

### Usage

To run Vale Server on your current document, select
Add-ons <i class="fas fa-long-arrow-alt-right"></i>
Vale <i class="fas fa-long-arrow-alt-right"></i> Show Sidebar:

<img alt="Google Docs menu" src="/vale-server/img/ui/Google/menu.png" style="width: 100%; display: block; margin: 0 auto;">

### Uninstallation

To remove the add-on, select
Add-ons <i class="fas fa-long-arrow-alt-right"></i>
Manage add-ons...:

<img alt="Google Docs menu" src="/vale-server/img/ui/Google/manage.png" style="width: 100%; display: block; margin: 0 auto;">

And then, next to the Vale add-on, select
MANAGE <i class="fas fa-long-arrow-alt-right"></i>
Remove:

<img alt="Google Docs menu" src="/vale-server/img/ui/Google/remove.png" style="width: 100%; display: block; margin: 0 auto;">
