Usage

There are two ways to use Vale Server: via the Check Clipboard as... menu option or through a third-party integration.

Clipboard

The Check Clipboard as... option allows you to use your Vale Server configuration anywhere that supports copy-and-paste (e.g., web browsers, email clients, content management systems, etc).
