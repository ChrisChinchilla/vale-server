Vale Server is a cross-platform (Windows and macOS) desktop application that
helps enforce writing style guides. It's built on top of, and offers a high
degree of compatibility with, the [Vale command-line tool](/vale/about).

## Activation

Usage of Vale Server requires a license key, which you can get from the [main website](https://errata.ai/vale-server/#puchase).
